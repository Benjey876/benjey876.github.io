About Me

My name is Benjamin Sherman. I am 25 years of age and live in Cardiff, South Wales. I graduated from University of Wales, Trinity Saint David in 2015 with a 1st Class Honours, BSc in Website Development. Since graduating, I have been hungry to expand my knowledge of the Web as well as improve my skills as both a developer and a person.

Portfolio

Check out some of my work below!

Mad Catz Support

Technologies used: HTML, CSS, JS, JQuery & Bootstrap

The Mad Catz support site was built inside the Kayako platform. It was my first project alongside the Tritton support page. I was tasked with the whole re-design and development of the pages. The site went through many different iterations until one was settled on, which was then brought to life. I continue to support and maintain this site day to day.

While working on the websites, I learnt the importance of cross browser compatibility as well as the need to make sure sites are responsive, allowing them to be viewed without affecting the user’s experience.

Tritton Support

Technologies used: HTML, CSS, JS, JQuery & Bootstrap

When I joined Mad Catz, this was a project completed alongside the recreation of the Mad Catz Support site. Initially, the Triton Support site co-existed within the Mad Catz Support site, but it was decided they would be split to have their own branded websites. This would create a unified experienced, and support each brand individually. I also continue to support and maintain this site day to day.

The Tritton Support site serves as a reminder as to what I can achieve, and how my skills have progressed from my university days where design was one of my weakest aspects.

Tritton Audio Centre

Technologies used: HTML, CSS, ASP.net, PHP & Bootstrap

Tritton Audio Centre was built as a portal for customers who had bought an Ark 100 for PC so they could download the software application that goes with it. The site was originally built in ASP.net but later had to be converted into PHP due to the new server configuration.

Owens Group Support

Technologies used: HTML, CSS & JS

Owens Group Remote Support page was built as a standalone site that would allow for the IT team of Owens Group to connect onto a user’s/customer’s machine when a code had been entered. This would allow the team to troubleshoot issues where the user may have little to no technical experience.
